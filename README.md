# Pasos para subir mi proyecto a Github

1. git init
2. Trabajamos en el proyecto (agregamos archivos, etc)
3. git add . (con el . agregamos TODOS los archivos)
4. git commit -m "mensaje" (Para confirmar los cambios)

Si no tengo repositorio remoto... 5. Creamos el repositorio en github 6. Seguimos los pasos desde git branch -m main 7. git rmeote add origin llink de repo 8. git push -u origin main
